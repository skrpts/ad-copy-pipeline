## UTM Parameter Convention

### Structure
`?utm_source={{source}}&utm_medium={{medium}}&utm_campaign={{campaign}}&utm_content={{content}}&utm_term={{term}}`

### Naming Rules
- All lowercase, hyphens for spaces
- source: platform name (google, facebook, linkedin, newsletter)
- medium: channel type (cpc, email, social, referral)
- campaign: campaign identifier (spring-sale-2026, product-launch-q2)
- content: variant identifier (hero-a, cta-blue, subject-line-2)
- term: keyword or audience (brand-terms, retargeting-30d)
