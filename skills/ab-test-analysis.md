## Capability

Analyses A/B and multivariate test results for statistical significance, practical impact, and actionable insights.

## When to Use

- Email subject line testing
- Landing page optimisation
- Ad creative testing
- Pricing experiments

## Inputs

Test setup (variants, hypothesis), results data (conversions, sample sizes, confidence intervals)

## Outputs

Winner recommendation, statistical significance assessment, effect size, next test suggestions
